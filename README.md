# 🦖 Kidszoo Academy

Kidszoo Academy is a modern, responsive landing page that showcases an interactive early childhood learning application for children aged **3–6 years**. The project features a fully functional mobile app simulator, allowing visitors to experience the application's educational modules directly from the website.

Designed with a child-friendly pastel color palette, smooth animations, and a clean user interface, Kidszoo combines playful visuals with interactive learning experiences to promote cognitive, literacy, numeracy, and visual development through gamification.
